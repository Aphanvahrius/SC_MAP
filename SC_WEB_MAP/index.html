ShipCore Interactive Map

An unofficial companion star-map for the ShipCore novel series. It charts the nations, star systems, hyperlanes, real-space distances and other information about the setting — and lets you filter and browse through it.

Using the map

  • Layers — the Layers tab toggles regions, star systems, special systems (mining, industrial, core, capital), hyperlanes, character journeys, diplomacy and system-name labels.
  • Filters — the Main and Other tabs filter by nation, book, character, star presence, orbitals/colonies, hyperlane type & status and more. A filter affects every layer that carries an associated field.
  • Books — some data is tagged by book volume (A1–A4). Switch the Book filter to see the map at that point in the story.
  • Diplomacy — enable the Diplomacy layer to colour nations by their relationship to one viewpoint nation. Double-click a nation on the map to switch the viewpoint to them.
  • Real Distances — in the Real Distances tab, pick a system to label every other system with its true 3-D distance from that origin (calculated based on real star coordinates).
  • Search — the magnifier (top-left, under the zoom) finds a system by name and moved the view to it. Click the icon again to close the text input field.
  • Details — click any feature for a popup; click empty space (or click the X icon) to dismiss it.
  • Fullscreen — the button at the top-right expands the map.

About ShipCore

ShipCore is a science-fiction LitRPG web-serial->novel series by Erios909. This map is a fan-made companion tool to help explore the setting.
